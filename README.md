# Sideboard
An online Magic: the Gathering card game play area.

![Lobby Screenshot](Sideboard/screenshots/ss1_lobby.png)

Web Server -
* NodeJS / ExpressJS
* Socket.io
* MongoDB
* Redis
* Passport

Client Side -
* AngularJS
* Socket.io-client
* HTML/CSS

Features -
* Login/Register
* Instant Status Update
* Lobby Chat / Game Chat
* Create Game / Change Game Settings
* Fullscreen
* etc...

In-progress -
* Deck Editor
* Game playable area
